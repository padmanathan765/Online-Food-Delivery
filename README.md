# Online Food Delivery Website

This project is a simple frontend-based Online Food Delivery Website built using HTML, CSS, and JavaScript.

## Features
- Browse food menu
- Add food items to cart
- Dynamic total price calculation
- Responsive layout for different screen sizes

## Technologies Used
- HTML5
- CSS3
- JavaScript
